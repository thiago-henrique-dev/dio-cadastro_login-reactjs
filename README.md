# Plataforma-da-Dio-React
Clone da Plataforma da dio com React + Typescript

<h2>Bibliotecas utilizadas</h2>
<li>ReactJS
<li>Styled-components
<li>React-hook-form
<li>React-router-dom
<li>json-server
<li>axios
<li>yup


  <h3>Instalando pacotes </h3>

`$ npm install` 
  
 <h3> Iniciar server <h3>
   
 `$ npm run server`
   
  <h3>Iniciar app <h3>
    
  `$ npm run dev`
